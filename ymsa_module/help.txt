(?) 帮助文档

(i) 说明
括号里的数字是警报代码，在警报弹窗里也有，您可以通过该代码快速寻找到文档中对应的段落
该文档只能尽力列出常见问题的原因和可能的解决方法
对于该文档没能预料到的问题，您应在必要时询问AI

(1) 导入Json文件失败
=> 触发条件
尝试读取「.\ymsa_module\user_config.json」时找不到该文件
=> 可能的原因
该文件不存在
=> 可能的解决方法
补上该文件

(2) 解析Json文件失败
=> 触发条件
尝试解析「.\ymsa_module\user_config.json」的内容时出错
=> 可能的原因
括号不成对、引号不成对
中间漏逗号、末尾多逗号
使用了全角标点符号
路径分隔符没有使用两个反斜杠「C:\\Users\\Yachiyo\\...」
在Json里写注释「// 就像这样」
=> 可能的解决方法
Json有着极其严苛的语法要求，建议优先把配置文件丢给AI询问
